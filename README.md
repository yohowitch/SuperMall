# supermall

### 1.简介
> * 基于vue-cli4搭建的前后端分离的商城项目
> * 包含商城主页，商品详情页，商品分类页，个人中心等模块
> * 熟悉使用Vue、Vue-Router、Vuex、Axios、Webpack等前端技术
> * 使用Better-Scroll更好地实现滚动效果
> * 通过Vue懒加载、防抖节流优化性能
> * 实现组件封装，模块化开发

> 3） 路由路径映射里是component不是components
